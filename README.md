# Apalabrados
Proyecto 1 : Desarrollo de Interfaces

### Desarollo de un juego basado en el juego Apalabrados o Wordle con las siguientes características: 
* 5 intentos.
* Usar palabras que existan de solo 5 letras.
* Palabra aleatoria en cada partida.
* Ir enseñando letras usadas, y pista de si la letra están en la palabra o en la posición.
* Mensaje resumen del nº de intentos usados, nº de letras utilizadas y palabras usadas.
* Si aciertas una palabra pasas a la siguiente hasta que se te acaben los intentos. 

Utilizando Javafx en IntelliJ y SceneBuilder con una base de datos o una lista de palabras.
